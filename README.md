🤖 AI Interview Agent (MERN Stack)

AI Interview Agent is a web-based platform that helps users practice mock interviews with AI-generated questions and get instant feedback to improve their skills.

🚀 Features
AI-powered interview questions
Instant answer feedback
User authentication (Login/Signup)
Save interview history
Responsive UI


🛠️ Tech Stack
React.js
Node.js
Express.js
MongoDB
OpenAI API

⚙️ Setup
git clone https://github.com/your-username/ai-interview-agent.git
cd ai-interview-agent
Frontend
cd client
npm install
npm start
Backend
cd server
npm install
npm start
